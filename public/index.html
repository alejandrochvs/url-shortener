URL Shortener

Part of the FreeCodeCamp | API projects.

By Alejandro Chaves.

How to add new URL.

Go to /new/[Insert your URL here], it has to be well formatted with [http://], it has to be a valid url like https://www.google.com/, this will return something like this: {"original_url":"https://www.google.com/","short_url":"http://localhost:3000/4990"}

How to go to existent URL.

Go to /[Insert short URL here], this will redirect you to the URL assigned to the short URL, for example: /4990.
